# scoop-bucket

Scoop bucket for [easonwumac](https://github.com/easonwumac)'s tools.

## Usage

```powershell
scoop bucket add easonwumac https://github.com/easonwumac/scoop-bucket
```

## Available

| Name | Description |
|------|-------------|
| `helm-host` | AI agent relay — monitor and control Codex sessions from your iPhone |

## Install

```powershell
scoop install easonwumac/helm-host
```
